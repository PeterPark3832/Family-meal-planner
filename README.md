# 🍽️ 우리 가족 식단 플래너

가족 구성원과 취향에 맞게 자동으로 식단을 추천해주는 웹 앱입니다.  
별도 설치 없이 브라우저에서 바로 사용할 수 있습니다.

👉 **[바로 사용하기](https://peterpark3832.github.io/Family-meal-planner/)**

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 👨‍👩‍👧‍👦 가족 구성원 설정 | 1~5명, 성인/자녀 구분, 성별·나이 입력 |
| 📅 기간 선택 | 1주 / 2주 / 3주 / 한 달 / 직접 입력(최대 90일) |
| 🍚 요리 종류 | 한식 / 양식 / 중식 / 일식 (중복 선택 가능) |
| 🎲 개별 재생성 | 마음에 안 드는 메뉴만 셀 호버 후 주사위 클릭으로 교체 |
| ✏️ 직접 입력 | 원하는 메뉴를 자유롭게 입력 + 메모 추가 |
| 📊 요리 분포 | 한식/양식/중식/일식 비율 시각화 |
| 📋 텍스트 복사 | 식단표를 텍스트로 클립보드에 복사 |
| 🖨️ 인쇄 | 깔끔한 레이아웃으로 인쇄 |
| 💾 자동 저장 | 브라우저 LocalStorage에 자동 저장, 다음 방문 시 복원 |

---

## 🧠 스마트 추천 로직

- **자녀 나이 기반 필터링**: 맵거나 자극적인 음식은 어린 자녀가 있을 경우 자동 제외
- **주간 중복 최소화**: 같은 주 내에서는 동일 메뉴가 반복되지 않도록 구성
- **균형 잡힌 식단**: 아침/점심/저녁 각 끼니에 맞는 메뉴풀에서 추천

---

## 🍽️ 메뉴 데이터베이스

총 **80여 가지** 집에서 해먹기 좋은 메뉴 수록

- **한식** (32종): 김치찌개, 된장찌개, 불고기, 갈비찜, 비빔밥, 삼겹살 외
- **양식** (19종): 크림파스타, 피자, 오므라이스, 카레라이스, 그라탕 외
- **중식** (12종): 짜장면, 탕수육, 마파두부, 유린기, 볶음밥 외
- **일식** (15종): 우동, 돈카츠, 오야코동, 라멘, 오코노미야키 외

## 🚀 로컬 실행

빌드 과정 없이 `index.html` 파일을 브라우저로 열기만 하면 됩니다.

```bash
# 파일 직접 열기
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```
