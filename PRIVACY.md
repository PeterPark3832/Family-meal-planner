## 1. 수집하는 개인정보

우리 가족 식단 플래너는 **서버를 운영하지 않으며**, 사용자의 개인정보를 수집·저장·전송하지 않습니다.

앱 사용 중 입력되는 다음 정보는 **사용자 기기의 브라우저 LocalStorage에만 저장**됩니다:

| 항목 | 저장 목적 | 저장 위치 |
|------|-----------|-----------|
| 가족 구성원 수, 나이, 성별 | 식단 생성 설정 복원 | 기기 LocalStorage |
| 선택한 요리 종류·기간·옵션 | 식단 생성 설정 복원 | 기기 LocalStorage |
| 생성된 식단 메뉴 목록 | 이전 식단 복원 | 기기 LocalStorage |
| 메뉴 좋아요/싫어요 평가 | 다음 추천 반영 | 기기 LocalStorage |
| 저장한 식단 템플릿 | 템플릿 불러오기 | 기기 LocalStorage |
| 직접 추가한 커스텀 메뉴 | 커스텀 메뉴 유지 | 기기 LocalStorage |

이 데이터는 **외부 서버로 전송되지 않으며**, 운영자는 해당 데이터에 접근할 수 없습니다.

---

## 2. 자동으로 수집되는 정보 (Google Analytics)

앱 품질 개선을 위해 **Google Analytics 4(GA4)**를 사용합니다.  
GA4는 다음과 같은 익명화된 사용 통계를 수집합니다:

- 페이지 방문 수, 세션 시간
- 어떤 기능이 얼마나 사용되었는지 (식단 생성 횟수, 공유 횟수 등)
- 사용 기기 유형 및 브라우저 종류
- 대략적인 지역 (시/도 수준, IP는 수집하지 않음)

수집되는 정보는 **개인을 식별할 수 없는 집계 데이터**입니다.  
GA4의 데이터 처리 방식은 [Google 개인정보처리방침](https://policies.google.com/privacy)을 참조하세요.

### GA4 수집 거부 방법

브라우저에 [Google Analytics 차단 확장 프로그램](https://tools.google.com/dlpage/gaoptout)을 설치하면 GA4 데이터 수집을 거부할 수 있습니다.

## 3. 쿠팡 파트너스 (광고)

장보기 목록 화면에는 쿠팡 파트너스 제휴 링크가 포함되어 있습니다.  
해당 링크를 통해 구매가 이루어지면 운영자에게 소정의 수수료가 지급될 수 있습니다.  
링크 클릭 시 쿠팡의 쿠키가 사용자 기기에 저장될 수 있으며, 이는 쿠팡의 개인정보처리방침을 따릅니다.

---

## 4. 카카오 공유 (선택 기능)

카카오 공유 기능을 사용하는 경우 Kakao JavaScript SDK가 로드되며,  
카카오의 개인정보처리방침에 따라 카카오 측에서 일부 정보를 수집할 수 있습니다.  
카카오 공유 버튼을 사용하지 않으면 SDK는 로드되지 않습니다.

---

## 5. 식단 URL 공유

"공유" 버튼을 누르면 현재 식단 설정과 메뉴 목록이 URL에 암호화(Base64) 인코딩되어  
공유 링크가 생성됩니다. 이 데이터는 서버에 저장되지 않으며, URL 링크를 받은 사람만 식단을 볼 수 있습니다.  
민감한 개인정보는 포함되지 않습니다 (가족 구성원 수, 나이 그룹, 메뉴 목록만 포함).

---

## 6. 데이터 삭제 방법

LocalStorage에 저장된 모든 데이터는 다음 방법으로 삭제할 수 있습니다:

- **브라우저 캐시/쿠키 삭제** (설정 → 인터넷 사용 기록 삭제)
- **개발자 도구 사용**: F12 → Application → Local Storage → 해당 항목 삭제

삭제 요청 시 별도로 운영자에게 연락하실 필요가 없습니다.

본 서비스는 만 14세 미만 아동으로부터 직접 개인정보를 수집하지 않습니다.  
아동의 나이·성별 정보는 보호자가 식단 생성을 위해 직접 입력하며, 서버에 저장되지 않습니다.
